# 📝 Todo App с голосовым управлением
Gриложение для управления задачами с поддержкой голосового ввода. Написано на React+JS с использованием Tailwind CSS для стилизации и Web Speech API для распознавания речи.

##  Особенности
- ✅ Создание, редактирование и удаление задач
- ✅ Отметка о выполнении
- ✅ Фильтрация задач (все/активные/завершенные) 
- ✅ 🎤 Голосовой ввод задач (на русском языке) (только Chrome и Edge)
- ✅ Drag-and-drop сортировка
- ✅ Сохранение задач и темы в `localStorage` и БД, с использованим кастомных хуков
- ✅ Синхронизация `localStorage` и БД
- ✅ Офлайн-режим (добавление и сохранение задач без интернета)

## 🛠️ Используемые технологии
| Категория | Технологии |
|-----------|-------------|
| **Фреймворк** | React (JS) |
| **Стилизация** | Tailwind CSS |
| **Drag-and-drop** | dnd-kit (`@dnd-kit/sortable`, `@dnd-kit/core`) |
| **Голосовой ввод** | Web Speech API (Speech Recognition) |
| **Хранение данных** | localStorage + База данных (mockApi) |
| **Управление состоянием** | Кастомные React хуки |
| **Синхронизация** | Двусторонняя синхронизация (localStorage ↔ БД) |
