# 📝 Todo App с голосовым вводом
Приложение для управления задачами с поддержкой голосового ввода. Написано на React+JS с использованием Tailwind CSS для стилизации и Web Speech API для распознавания речи.

##  Особенности
- ✅ Создание, редактирование и удаление задач
- ✅ Отметка о выполнении
- ✅ Фильтрация задач (все/активные/завершенные) 
- ✅ 🎤 Голосовой ввод задач (на русском языке)
- ✅ Drag-and-drop сортировка
- ✅ Сохранение задач и темы в `localStorage` и БД, с использованим кастомных хуков
- ✅ Синхронизация `localStorage` и БД

## 🛠️ Используемые технологии
| Категория | Технологии |
|-----------|-------------|
| **Фреймворк** | React (JS) |
| **Стилизация** | Tailwind CSS |
| **Drag-and-drop** | (`@dnd-kit/react`) |
| **Голосовой ввод** | Web Speech API |
| **Хранение данных** | localStorage + База данных (mockApi) |
| **Управление состоянием** | Кастомные React хуки |
