# 📝 Todo App с голосовым управлением
Gриложение для управления задачами с поддержкой голосового ввода. Написано на React+JS с использованием Tailwind CSS для стилизации и Web Speech API для распознавания речи.

##  Особенности
- ✅ Создание, редактирование и удаление задач
- ✅ Отметка о выполнении
- ✅ Фильтрация задач (все/активные/завершенные) 
- ✅ 🎤 Голосовой ввод задач (на русском языке)
- ✅ Drag-and-drop сортировка
- ✅ Сохранение задач и темы в `localStorage` и БЗ, с использованим кастомных хуков
- ✅ Синхронизация `localStorage` и БЗ
- ✅ Возможность добавления и сохранения задач в офлайн режиме
